# Category Cards Block

Displays a section heading with a grid or carousel of category cards.

## Variants

- **category-cards machine** — three-column grid for machine categories
- **category-cards carousel** — horizontal carousel for featured categories

## Content Structure

| Category Cards (machine) |
|--------------------------|
| ## Section Title |
| Section description |
| ![image](image.jpg) | [Category Name](/path) |

Each card row contains an image column and a link column. The first rows without images are treated as the section header.
