# Product Cards Block

Displays product cards with image, title, SKU, rating, description, and price link.

## Variants

- **product-cards carousel** — horizontal scrollable product row with navigation arrows

## Content Structure

| Product Cards (carousel) |
|--------------------------|
| ## Section Title |
| Section description |
| ![image](image.jpg) | **Product Title** | **SKU:** ABC123 | (6) | Description text | [Login to see the price](/login) |

Each product row uses up to six columns: image, title, SKU, rating count, description, and price link.
