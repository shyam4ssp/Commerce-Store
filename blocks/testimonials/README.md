# Testimonials Block

Customer testimonial carousel with navigation and optional feedback CTA.

## Content Structure

| Testimonials |
|--------------|
| ## Section Title |
| Section description |
| J | **Name** / Title | "Quote text" |
| [Submit your feedback](/feedback) |

Each testimonial row has three columns: avatar initial, author details, and quote. The final row with a button link becomes the section CTA.
